🏥 Healthic - Sistema de Gestión de Instrumental Médico

Digitaliza el proceso completo de esterilización hospitalaria con trazabilidad total y cumplimiento normativo.

🎯 Funcionalidades Principales

📋 Gestión de Órdenes

  • Dashboard en tiempo real con estado de cada orden
  • Numeración automática (MS2-001, MS2-002...)
  • Transiciones de estado validadas
  • Botones de acción rápida por etapa

🔬 Control de Procesos

  • Registro de métodos de lavado y esterilización
  • Validación de compatibilidad instrumento-método
  • Control de temperaturas y tiempos
  • Verificación de cincho verde

📊 Trazabilidad Completa

  • Seguimiento individual por instrumento
  • Registro de responsables por proceso
  • Evidencias fotográficas y documentales
  • Reportes por cliente, fecha y tipo

🏢 Gestión de Clientes

  • Configuración de tipos de facturación (STU, Metro³, Por Carga)
  • Información de contacto y direcciones
  • Historial de órdenes por cliente

📱 Optimizado para Operadores

  • Interfaz tablet-friendly para áreas de lavado, gris y blanca
  • Widgets especializados: toggles, radio buttons, badges de estado
  • Validaciones visuales con campos obligatorios destacados
  • Accesos rápidos para tareas frecuentes
  • Botones grandes con iconos intuitivos

⚙️ Configuración Técnica

  • Modelos: 5 modelos principales con relaciones optimizadas
  • Permisos: Control diferenciado por tipo de usuario
  • Datos iniciales: 8 métodos preconfigurados + datos demo
  • Validaciones: 15+ validaciones de negocio implementadas
  • Cálculos automáticos: STU, duraciones, estadísticas

🚀 Instalación y Uso

  1. Instalar el módulo desde Apps
  2. Configurar métodos de lavado/esterilización (precargados)
  3. Registrar clientes hospitalarios
  4. Crear catálogo de instrumentos
  5. ¡Comenzar a procesar órdenes!
